### Prerequisites

- Flutter SDK (latest version)
- Dart SDK

### Installation

1. Clone the repository
```bash
git clone https://github.com/mamipro-123/flutter-admin-panel.git
```

2. Navigate to project directory
```bash
cd admin_panel
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
